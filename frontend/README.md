# Mini Duolingo (Frontend)

Lightweight Vite + React + TypeScript frontend for the microservices platform.

Run:

```bash
cd frontend
npm install
npm run dev
```

Set the API gateway URL via `VITE_API_URL` (default `http://localhost:3000/api/v1`):

```bash
VITE_API_URL=http://localhost:3000/api/v1 npm run dev
```
